# 🎬 Flixbuzz Multi-Platform Workspace

এই ওয়ার্কস্পেসে (Workspace) আপনার তৈরি করা **ফ্লিক্সবাজ (Flixbuzz)** অ্যাপের অ্যান্ড্রয়েড সোর্স কোড, ওয়েব ক্লায়েন্ট এবং ব্যাকএন্ড সার্ভার কোড সুবিন্যস্ত অবস্থায় রয়েছে।

## 📁 প্রজেক্ট স্ট্রাকচার (Project Structure)
1. **/app**: আধুনিক Jetpack Compose ও Kotlin দিয়ে তৈরি সম্পূর্ণ অ্যান্ড্রয়েড অ্যাপ্লিকেশন।
2. **/web-app**: লাইভ স্ট্রিমিং ও ক্রিয়েটর পোর্টালের সিঙ্গেল-পেজ রেসপনসিভ ওয়েব অ্যাপ্লিকেশন (HTML + TailwindCSS + Vanilla JS)। যা সরাসরি MongoDB-তে রিয়েল-টাইমে ডেটা সিঙ্ক করতে পারে।
3. **/backend-server**: Node.js, Express দিয়ে তৈরি ব্যাকএন্ড সার্ভার। যা ডুডস্ট্রিম রিমোট আপলোড হ্যান্ডল করার জন্য এপিআই প্রক্সি হিসেবে কাজ করে।

## 🚀 গিটহাব এবং নেটলিফাই (Netlify) ডিপ্লয়মেন্ট গাইড

যেহেতু আপনি এই সম্পূর্ণ রিপোজিটরিটি গিটহাবে পুশ করে নেটলিফাই-তে ডিপ্লয় করবেন, তাই নিচের ধাপগুলো অনুসরণ করুন যেন কোনো কনফিগারেশন মিস না হয়:

### 1. গিটহাবে পুশ করা (Pushing to GitHub)

আপনি মোবাইল অথবা কম্পিউটার থেকে খুব সহজেই এটি গিটহাবে আপলোড করতে পারেন:

#### ক) গিট কমান্ড লাইনের মাধ্যমে (টার্মিনাল/পিসি বা Termux থেকে):
```bash
# গিট ইনিশিয়ালাইজ করুন
git init

# সব ফাইল গিটে যোগ করুন
git add .

# কমিট লিখুন
git commit -m "Initial commit for Flixbuzz Multi-Platform Workspace"

# আপনার নতুন গিটহাব রিপো লিংক যুক্ত করুন
git remote add origin https://github.com/YOUR_USERNAME/flixbuzz-workspace.git

# মেইন ব্রাঞ্চ পুশ করুন
git branch -M main
git push -u origin main
```

#### খ) ব্রাউজারের মাধ্যমে সরাসরি ড্র্যাগ অ্যান্ড ড্রপ (কোনো কোডিং ছাড়া):
১. এই প্রজেক্টের সাইডবার থেকে সম্পূর্ণ প্রজেক্টের জিপ (ZIP) ফাইলটি ডাউনলোড করুন (**Settings > Export Project as ZIP**)।
২. জিপ ফাইলটি আপনার মোবাইলে বা পিসিতে আনজিপ করুন।
৩. [GitHub.com](https://github.com) এ লগইন করে একটি নতুন রিপোজিটরি খুলুন।
৪. ব্রাউজারে **"uploading an existing file"** অপশনে ক্লিক করুন।
৫. আপনার ফাইলগুলো টেনে এনে বা নির্বাচন করে আপলোড করে দিন!

---

### 2. Netlify তে ওয়েব অ্যাপ ডিপ্লয়মেন্ট (Deploying to Netlify)

যেহেতু আমাদের ওয়েব অ্যাপটি `/web-app` ডিরেক্টরিতে অবস্হিত, নেটলিফাই স্ক্রীন থেকে কানেক্ট করার সময় নিচের সেটিংসগুলো ব্যবহার করবেন:

১. **Netlify Dashboard** এ যান এবং **"Add new site" > "Import from Git"** এ ক্লিক করুন।
২. আপনার গিটহাব অ্যাকাউন্ট কানেক্ট করে এই রিপোজিটরিটি সিলেক্ট করুন।
৩. **Site Configuration / Build Settings** এ নিচের তথ্যগুলো দিন (এটি অত্যন্ত গুরুত্বপূর্ণ):
   - **Base directory:** `web-app`  *(এর ফলে নেটলিফাই শুধুমাত্র ওয়েব অ্যাপ ডিরেক্টরি বিল্ড করবে)*
   - **Build command:** *(ফাঁকা রাখুন, কিছুই লিখতে হবে না)*
   - **Publish directory:** `.` *(যেহেতু বেস ডিরেক্টরি `web-app` সিলেক্ট করা আছে, তাই ডট চিহ্নই যথেষ্ট)*
৪. **Deploy site** এ ক্লিক করুন। কয়েক সেকেন্ডের মাঝেই আপনার চমৎকার ওয়েব পোর্টালটি লাইভ হয়ে যাবে!

### 3. Render.com এ ব্যাকএন্ড এক্সপ্রেস সার্ভার ডিপ্লয়মেন্ট (Optional)

যদি আপনি কাস্টম ব্যাকএন্ড বা এপিআই প্রক্সি ব্যবহার করতে চান:
১. [Render.com](https://render.com) এ লগইন করে **New Web Service** সিলেক্ট করুন।
২. এই গিটহাব রিপোজিটরিটি কানেক্ট করুন।
৩. কাস্টমাইজেশন সেটিংসে নিচের কনফিগারেশন দিন:
   - **Root Directory:** `backend-server`
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
৪. **Advanced** এ গিয়ে Environment Variables যোগ করুন (যেমন: `MONGO_APP_ID`, `MONGO_API_KEY`, `DOOD_API_KEY` যার ডিফল্ট মান আপনার কোডে দেওয়া আছে)।
৫. **Deploy** বাটনে প্রেস করুন।
