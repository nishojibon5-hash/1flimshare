# 🎬 Flixbuzz Web App & Express Backend — 0 Cost Deployment Guide (Bangla)

এই ডিরেক্টরিতে আপনার তৈরি করা ফ্লিক্সবাজ (Flixbuzz) স্ট্রিমিং অ্যাপের সম্পূর্ণ **ওয়েব ক্লায়েন্ট (Frontend)** এবং **এক্সপ্রেস ব্যাকএন্ড সার্ভার (Backend)** কোড রয়েছে। আপনি প্রিমিয়াম ফিচারসমূহ সহ সম্পূর্ণ সিস্টেমটি ১০০% ফ্রিতে আজীবনের জন্য ওয়েব হোস্টিং প্ল্যাটফর্মে লাইভ করতে পারবেন।

---

## ⚡ ১. ওয়েব অ্যাপ ডিপ্লয়মেন্ট (Netlify তে ফ্রিতে হোস্টিং - ০ টাকা)

Netlify খুব সহজে স্ট্যাটিক ওয়েব পেজ হোস্টিং দেয়। যেহেতু আমাদের ফ্রন্টএন্ড ক্লায়েন্ট সম্পূর্ণ একক-পৃষ্ঠা (Single Page Layout) এবং সরাসরি মঙ্গোডিবি ও ডুডস্ট্রিম কানেক্ট করতে পারে, তাই কোনো জটিল বিল্ড ছাড়াই এটি সরাসরি ড্র্যাগ-এন্ড-ড্রপ পদ্ধতিতে বা গিটহাবের মাধ্যমে ফ্রিতে হোস্ট করা সম্ভব।

### ক) সরাসরি ড্র্যাগ-এন্ড-ড্রপ পদ্ধতি (সবচেয়ে সহজ এবং ১ ক্লিক উপায়):
১. এই প্রজেক্টের `/web-app` ফোল্ডারটি জিপ (ZIP) করুন।
২. [Netlify App Drop](https://app.netlify.com/drop) পেজে যান।
৩. সেখানে ব্রাউজারে জিপ ফাইলটি টেনে এনে ছেড়ে (Drag and Drop) দিন।
৪. ১০ সেকেন্ডের মাঝেই আপনার সাইটটি লাইভ হয়ে যাবে এবং একটি পাবলিক লিংক পাবেন (যেমন `https://flixbuzz-portal.netlify.app`)!

### খ) গিটহাবের (GitHub) মাধ্যমে অটো-ডিপ্লয় পদ্ধতি:
১. একটি ফ্রি [Github Account](https://github.com) খুলুন।
২. আপনার গিটহাব অ্যাকাউন্টে একটি নতুন রিপোজিটরি (e.g. `flixbuzz-web-portal`) তৈরি করুন।
৩. এই `/web-app` ফোল্ডারের ফাইলসমূহ (index.html, app.js, styles.css, netlify.toml) পুশ করুন।
৪. [Netlify Dashboard](https://app.netlify.com) এ লগইন করে **"Add new site" > "Import from Git"** এ ক্লিক করুন।
৫. আপনার গিটহাবের ওই রিপোজিশন সিলেক্ট করুন এবং **"Deploy site"** বাটনে প্রেস করুন। এখন গিটহাবে কোনো কোড চেঞ্জ করলে নেটলিফাই স্বয়ংক্রিয়ভাবে সেটি নতুন করে আপডেট করে দেবে।

## 🌐 ২. এক্সপ্রেস ব্যাকএন্ড সার্ভার ডিপ্লয়মেন্ট (Render বা Koyeb তে ফ্রিতে ০ টাকা হোস্টিং)

যদি আপনি কাস্টম সার্ভার রাউট ব্যবহার করতে চান এবং ক্লায়েন্ট থেকে মঙ্গোডিবি এবং ডুডস্ট্রিমের সিক্রেট কি লুকাতে চান, তবে এক্সপ্রেস সার্ভারটি হোস্টিং করা প্রয়োজন। ফ্রি ও সহজে রাডারে হোস্ট করার নিয়ম নিচে দেওয়া হলো:

### Render.com ব্যবহার করে হোস্টিং:
১. একটি সম্পূর্ণ ফ্রি একাউন্ট খুলুন [Render.com](https://render.com) এ।
২. আপনার গিটহাবে একটি নতুন রিপোজিটরি বানিয়ে এই `/backend-server` ফোল্ডারের সমস্ত কন্টেন্ট সেখানে আপলোড বা পুশ করুন।
৩. Render ড্যাশবোর্ডে গিয়ে **"New +" -> "Web Service"** অপশনটি নির্বাচন করুন।
৪. আপনার গিটহাব একাউন্ট কানেক্ট করে ব্যাকএন্ড সার্ভারের রিপোজিটরি নির্ধারণ করুন।
৫. নিচের কনফিগারেশন বসান:
   - **Language:** `Node`
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Instance Type:** `Free` ($0/month)
৬. নিচে **"Advanced"** এ ক্লিক করে এনভায়রনমেন্ট ভ্যারিয়েবল (Environment Variables) যোগ করুন:
   - `MONGO_APP_ID` = `data-flixbuzz-pqrst`
   - `MONGO_API_KEY` = `65f012cf01bcdc6179abcf9e123abcde`
   - `DOOD_API_KEY` = `569149jtwoitkto1zmwf1s`
৭. **"Create Web Service"** বাটনে ক্লিক করুন। ২-৩ মিনিটের মাঝে আপনার এক্সপ্রেস সার্ভার ইন্টারনেটে লাইভ হয়ে যাবে ও একটি লিংক পাবেন (যেমন: `https://flixbuzz-server.onrender.com`)।

## 🛠️ ৩. মোবাইল দিয়ে কোড সম্পাদনা বা ডাউনলোড করার উপায় (আমার পিসি নেই)

যেহেতু আপনার কাছে বর্তমানে পিসি নেই, তাই ফোন বা ট্যাবলেট থেকেই আপনি সম্পূর্ণ কাজ করতে পারবেন:
- **Zip ডাউনলোড:** এই ডেক্সটপ এডিটরটির একদম উপরে বা সেটিংস মেনু থেকে **"Export Project as ZIP"** এ ক্লিক করে সম্পূর্ণ জিপ ফাইলটি আপনার মোবাইলের মেমরিতে ডাউনলোড করুন।
- **জিপ আনজিপ:** মোবাইলের ফাইল ম্যানেজার বা ZArchiver অ্যাপ দিয়ে ফাইলগুলো আনজিপ করুন।
- **মোবাইলে GitHub আপলোড:** মোবাইলের ব্রাউজারটিকে 'Desktop Site' মোড সিলেক্ট করে খুব সহজে ফাইল ড্রাগ এন্ড ড্রপ পদ্ধতিতে গিটহাবে নতুন কোড আপলোড করতে পারেন। এছাড়া প্লে-স্টোর থেকে 'Termux' বা 'Pocket Git' ব্যবহার করেও পুশ করা সম্ভব।
- **লাইভ সেটিংস এডিট:** চ্যানেল প্রোফাইল বা নতুন ভিডিও আপলোডের সম্পূর্ণ কাজ কোনো কোডিং ছাড়াই আপনি সরাসরি মোবাইলে ব্রাউজারের নেভিগেটর বা আমাদের তৈরি করা প্রিমিয়াম ওয়েব ড্যাশবোর্ড থেকেই লাইভ ইন্টারনেটের মাধ্যমে সম্পন্ন করতে পারবেন!
